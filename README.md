# AI Video Studio

A complete, production-ready AI Video Generation and Editing application built with Flutter. This cross-platform app provides a modern, responsive UI similar to leading AI platforms.

## Features
- **Dark Mode Interface**: Modern slate and indigo theme.
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop/web.
- **Video Generation**: Text-to-video prompt interface.
- **Dashboard**: Recent project management and quick actions.
- **Cross-Platform**: Runs natively on iOS, Android, macOS, Windows, Linux, and Web.

## Tech Stack
- Frontend: Flutter
- UI: Material 3, custom responsive layouts
- State/Routing: Flutter native routing

## Setup Instructions
1. Install Flutter (https://docs.flutter.dev/get-started/install)
2. Run `flutter pub get`
3. Start the application: `flutter run`
